# Dillinger

[![N|Solid](https://cldup.com/dTxpPi9lDf.thumb.png)](https://nodesource.com/products/nsolid)

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

Planetary API stores information about planets and a few interesting units such as their

- mass
- radius
- distance from earth

#Features!

- Add planets
- Edit planet data
- delete
- diplay all planets
- display a single planet

It uses flask framework and also also:

- Flask-Mail
- Flask-JWT
- Flask Marshmallow
- SQL Alchemy
- SQLite
-

### Tech

Dillinger uses a number of open source projects to work properly:

- [PyCharm] - PyCharm Editor
- [Flask] - Flask
